Course 1

Our 14 week evening course is designed for those who would like to learn more flexibly in their own time.

Key Information

Start dates: 18th April 2016 Osterley (London), 1st May 2016 Leeds

On this course you’ll learn the basics of Software Development using modern web technologies.

For the first week you’ll study full time at the Sky Campus in your chosen location (London or Leeds). You’ll then come into the Sky Campus on one evening each week, and will also have a remote lesson via Skype once a week. You’ll also need to do a few hours of work each week on your own between lessons.

By the end of this course you should have a strong understanding of the basics of modern Web Development, and will have developed some small case study projects of your own. You should be equipped with the skills you need to start developing web applications from scratch.

The course is designed to give you enough technical skills to pass one of our entry level Assessment Centres, and to apply for a Software Developer role with us if you want to.

Modules

On this course you’ll learn the basics of web development, covering topics such as:
  • The Internet & Its Technologies
  • Object Oriented Programming
  • Databases
  • Security
  • HTML & CSS
  • JavaScript
  • PHP
  • MySQL

Important Dates

The first evening courses start 18th April 2016 in London, 1st May 2016 in Leeds
